# Carbon-Webapi
A Webapi that make you can easily get Carbon image.

# How To Use

1. Download Firefox
2. Edit Appsettings.json and put the path of firefox and the max retry

# Api

```
[post]
EndPoint:/CarbonPng
JsonBody:
{
	"Theme":"seti",
    "CodeType":"c++",
    "Code":"#include <iostream>\nint main()\n{\n}"
}
```
